Queue Data Structure

A Queue is a linear data structure that follows the First In First Out (FIFO) principle, meaning the first element added to the queue will be the first one to be removed. It supports two main operations: enqueue (adding an element to the rear of the queue) and dequeue (removing an element from the front of the queue). Queues are commonly used in scenarios such as task scheduling, breadth-first search in graphs, and handling requests in operating systems.

Time Complexity:

Enqueue Operation: O(1) (constant time to add an element to the rear of the queue).
Dequeue Operation: O(1) (constant time to remove an element from the front of the queue).
Peek Operation: O(1) (constant time to view the front element without removing it).

Space Complexity:

O(n), where n is the number of elements in the queue. The space is proportional to the number of elements stored in the queue.

Queues are a fundamental data structure used extensively in computer science. Variants of queues, such as circular queues, priority queues, and double-ended queues (deques), provide additional flexibility for specialized use cases.
