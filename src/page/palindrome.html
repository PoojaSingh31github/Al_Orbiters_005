Valid Palindrome

The Valid Palindrome problem asks us to determine whether a given string is a palindrome. A string is considered a palindrome if it reads the same forward and backward, after ignoring spaces, punctuation, and case sensitivity.

Time Complexity:

O(n), where n is the length of the string

Space Complexity:

O(n) for the space used to store the sanitized version of the string.

For most palindrome checks, spaces, punctuation, and case are ignored. For example, "A man, a plan, a canal, Panama!" is considered a palindrome when spaces, commas, and punctuation are removed, and the case is disregarded. Result: "amanaplanacanalpanama",which is a Palindrome
