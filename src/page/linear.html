Linear Search Algorithm

Linear Search is a simple algorithm that searches for a target element in an unsorted or sorted array by checking each element sequentially from the beginning until the target is found or the entire array has been traversed. Unlike binary search, it does not require the data to be sorted.

Time Complexity:

- Best Case: O(1) (when the target is the first element).
- Average Case: O(n) (on average, half of the elements need to be checked).
- Worst Case: O(n) (when the target is the last element or not present).

Space Complexity:

- Linear Search: O(1) (it requires constant space as it operates in place).

Linear Search is a straightforward approach to searching, making it ideal for small or unsorted datasets where other more complex algorithms like binary search would not be necessary. However, for large datasets, its performance may not be as efficient as other algorithms.
