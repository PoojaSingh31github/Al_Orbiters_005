String Data Structure

A String is a sequence of characters, typically used to represent text. Strings are one of the most commonly used data structures in programming, enabling the manipulation and storage of textual data. Strings are immutable in many programming languages, meaning any modification creates a new string. Operations such as concatenation, substring extraction, and character replacement are widely supported.

Time Complexity:

- Access by Index: O(1) (constant time to access a character).
- Concatenation: O(n), where n is the length of the new string.
- Substring Extraction: O(k), where k is the length of the substring.
- Search: O(n), where n is the length of the string being searched.
- Replace/Modify: O(n), as a new string must be created in most cases.

Space Complexity:

- The space complexity of a string depends on its size. For a string of length n, space complexity is O(n) for ASCII and O(2n) for Unicode. - Operations like concatenation may require additional memory allocation proportional to the resulting string's length.

Strings are foundational in programming, enabling everything from basic text processing to complex algorithms like pattern matching, encryption, and data serialization. They are extensively used in file manipulation, web development, and natural language processing (NLP).
